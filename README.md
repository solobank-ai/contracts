# Solobank Contracts

Solana programs for Solobank -- an AI bank account for autonomous agents.

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

> **Status**: This repo is a placeholder for upcoming Anchor smart contracts.

## Planned Programs

### AI Decision Vault

On-chain program for agent-managed vaults with configurable safeguards:

- Spending limits and cooldown periods
- Multi-sig approval for large transactions
- Automated yield rebalancing rules
- Emergency lockdown triggers

## Tech Stack (Planned)

- **Anchor** framework
- **Solana** mainnet / devnet
- Rust
